# Usage

## Running the Project
To start the development server, run:
```sh
    npm start
    # or
    yarn start
```

## Running on Android
```sh
    npm run android
    # or
    yarn android
```

## Running on iOS
```sh
    npm run ios
    # or
    yarn ios
```

## Running on Web
```sh
    npm run web
    # or
    yarn web
```

## Building for Production
To create a production build, run:
```sh
    expo build:android
    # or
    expo build:ios
```
